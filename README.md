# 🏦 城市智能房产估值与信贷风控平台 (Urban Property Risk System)

本项目是一个基于 Python 手写实现的端到端金融风控集成系统。它结合了房产抵押估值与信贷风险评估，涵盖了从底层矩阵算法推导到可视化 Web 应用的全流程开发。

## 🌟 项目亮点
* **算法原生实现**：拒绝仅通过 `sklearn` 调包，手动实现了多元线性回归的正规方程解、逻辑回归的梯度下降、LDA 线性判别分析等核心算法。
* **业务逻辑联动**：实现了房产估值（M1）与信贷风控（M2）的动态联动，通过抵押率 (LTV) 实时影响风险决策。
* **工业级可视化**：基于 Streamlit 搭建了包含房产估值、信贷审批、性能看板、模型对比四个板块的交互式工作台。
* **不平衡处理优化**：通过代价敏感学习（阈值移动），将“坏人”识别召回率提升至 **86%**。

## 📂 模块说明
按照项目规范，系统划分为以下核心模块：
* `src/m1_matrix_solve.py`: 房产估值模型，通过正规方程求解 $w^* = (X^TX)^{-1}X^Ty$。
* `src/m2_logistic_regression.py`: 信贷违约预测，手写 Sigmoid 函数与对数似然损失。
* `src/m3_lda_dimension.py`: 使用 LDA 进行特征降维与可视化。
* `src/m4_multi_classification.py`: 基于 OvO 策略的风险等级 (A/B/C/D) 评定。
* `src/m5_imbalance_handling.py`: 类别不平衡处理，实现坏人精准识别。
* `app.py`: 系统集成入口，提供可视化 UI 界面。
